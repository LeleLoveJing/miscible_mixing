Detailed Description

template<int dim, int data_dim>
class ps_mmm::Particle::DataParticle< dim, data_dim >

DataParticle provides an example of how to extend the BaseParticle class to include related particle data.

This allows users to attach scalars/vectors/tensors/etc to particles and ensure they are transmitted correctly over MPI and written to output files.
