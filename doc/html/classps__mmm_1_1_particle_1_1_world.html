template<int dim, class T>
void ps_mmm::Particle::World< dim, T >::send_recv_particles ( )
inline

Transfer particles that have crossed subdomain boundaries to other processors.

Because subdomains can change drastically during mesh refinement, particle transfer occurs as follows: - Each subdomain finds the particles it owns which have fallen outside it - For each particle outside the subdomain, send the particle to all subdomains and let them determine which one owns it. This assumes there is no overlap between subdomains. - Each process determines which of the received particles is in its subdomain, keeps these and deletes the others - TODO: handle particles outside any domain - TODO: if we know the domain of a particle (e.g. bordering domains), send it only to that domain
