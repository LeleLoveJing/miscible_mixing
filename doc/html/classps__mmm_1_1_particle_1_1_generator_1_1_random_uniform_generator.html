Generate a set of particles uniformly randomly distributed within the specified triangulation.

This is done using "roulette wheel" style selection weighted by cell volume. We do cell-by-cell assignment of particles because the decomposition of the mesh may result in a highly non-rectangular local mesh which makes uniform particle distribution difficult.

Parameters
[in]worldThe particle world the particles will exist in
[in]num_particlesThe number of particles to generate in this subdomain
[in]start_idThe starting ID to assign to generated particles
